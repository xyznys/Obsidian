##### 向量数据库
1. 安装
`pip3 install pandas sentence-transformers faiss-cpu openpyxl scikit-learn`
2. pandas - 处理Excel / csv，数据处理和分析库
3. sentence-transformers - 句子嵌入（Sentence Embeddings）模型库，用来做 embedding
4. FAISS - 用来做向量检索，Facebook AI开发的向量相似度搜索库（CPU版本）
5. openpyxl - Excel文件(.xlsx)读写库
6. scikit-learn - 机器学习库，计算相似度（辅助）
