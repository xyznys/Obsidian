
##### 背景：
一个由 **Microsoft** 推出的浏览器自动化工具，本质上是一个“用代码控制浏览器”的框架。你可以把它理解为：比 Selenium 更现代、更强大的一代浏览器自动化解决方案

##### 前置安装条件
[[Node.js]]
	playwright官方最强支持，教程最多，生态最好

##### 安装
Node 版
`npm init -y`
`npm install playwright`
`npx playwright install`

驱动语言：
1. JavaScript / TypeScript
2. Python

PlayWirght 相当于是一个编程库 Library
